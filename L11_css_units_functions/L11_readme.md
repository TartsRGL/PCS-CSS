# Lesson 11 -- CSS Units & Functions

This folder contains the HTML & CSS code for **Lesson 11** of the
*PCScss* series.

It demonstrates:

-   The use of modern **CSS units**: `rem`, `em`, `vw`, `vh`, `svh`.
-   Responsive layout with **math functions**: `clamp()`, `min()`,
    `max()`.
-   Spacing and typography scaling without media queries.
-   A fully responsive **cards grid** using `minmax()` and `auto-fit`.
-   A hero section with **fluid viewport height** (`vh`/`svh`).
-   Reusable **utility classes** (`.btn`, `.fluid`, `.text-center`,
    `.mt-*`).
-   Progressive enhancement with visual polish.

Each commit adds one conceptual layer step-by-step.
